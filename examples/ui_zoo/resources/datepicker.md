# DatePicker

The `DatePicker` widget allows users to select a date from a calendar view.

## Features
-   **Month Navigation**: Navigate between months using previous/next buttons.
-   **Date Selection**: Click on a day to select it.
-   **Leap Year Support**: Correctly handles leap years.

## Usage
```rust
<DatePicker> {
    width: Fit, height: Fit
}
```

## Styling
The widget uses standard theme colors but can be customized via `draw_bg` and other properties.
